SS1 Week 2: Technical Drawing Instruments & Materials

Learning Objectives

By the end of this lesson, students should be able to identify and state the uses of basic technical drawing instruments.

1. Introduction to Drawing Instruments

Technical drawing requires precision. To achieve this, we use special instruments designed for accuracy. The main instruments every student must have are shown below.

2. Essential Drawing Instruments and Their Uses

Drawing Board: A flat board to place your paper on.

T-square: Used for drawing horizontal lines and as a guide for set squares.

Set Squares: Used for drawing vertical and inclined lines, usually 45° and 60°/30°.

Compass: Used for drawing circles and arcs.

Divider: Used for transferring measurements and dividing lines.

Protractor: Used for measuring and marking angles.

Note: Always handle your instruments with care. A damaged T-square or set square will give wrong results.
