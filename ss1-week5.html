Dimensioning & Scale Drawing

Learning Objectives: By the end of this lesson, students should be able to apply basic rules of dimensioning and understand how to use scales to reduce or enlarge drawings.

1. Dimensioning – Basic Rules and Principles

Overview

Dimensioning is the process of adding measurements, notes, and specifications to a technical drawing so that a craftsman or engineer can produce the object exactly as designed.

Types of Dimensioning

A. Linear Dimensioning

  • Measures straight distances
  • Horizontal, vertical, or aligned

B. Angular Dimensioning

  • Measures angles between two lines
  • Shown in degrees (°)

C. Radial Dimensioning

  • Radius (R) – Used for arcs
  • Diameter (⌀) – Used for full circles

D. Ordinate Dimensioning

  • Uses X and Y coordinates from a datum point
  • Common in CNC machining

E. Chain Dimensioning

  • Dimensions placed end-to-end in a chain
  • Errors can accumulate

F. Parallel/Baseline Dimensioning

  • All dimensions measured from one baseline/datum
  • More accurate than chain dimensioning

Basic Rules of Dimensioning

  1. Every drawing must have enough dimensions — not too few, not too many
  2. Dimensions should not be repeated
  3. Dimension lines should not cross each other if avoidable
  4. Extension lines should not cross dimension lines
  5. Dimensions should be placed outside the object when possible
  6. The larger dimensions go on the outside, smaller ones inside
  7. All dimensions are in the same unit (usually mm)
  8. Dimension figures are placed above the dimension line
  9. Arrowheads must be neat, sharp, and consistent
  10. Centre lines should not be used as dimension lines

Elements of a Dimension

  • Dimension Line – Thin line with arrowheads
  • Extension Lines – Project from the object to the dimension line
  • Arrowheads – Mark the limits of the dimension
  • Dimension Figure – The actual measurement value
  • Leader Line – Points to a specific feature with a note

Advantages of Proper Dimensioning

  • Ensures the object is made correctly
  • Eliminates ambiguity
  • Speeds up the manufacturing process
  • Essential for quality control

Disadvantages of Poor Dimensioning

  • Leads to manufacturing errors
  • Causes confusion between teams
  • Results in costly rework
  • Can compromise safety

2. Scale Drawing – Introduction & Types

Overview

A scale drawing is a drawing where the dimensions of the object are reduced or enlarged by a consistent ratio so the object can fit on a drawing sheet while maintaining accurate proportions.

What is Scale?

Scale = Drawing Size : Actual Size

Written as a ratio, e.g., 1:10 means 1 unit on paper = 10 units in reality.

Types of Scale

A. Full Scale (1:1)

  • Drawing is the same size as the actual object
  • Used for small components like bolts, nuts, brackets

B. Reduction Scale

  • Object is larger than the drawing
  • Examples: 1:2, 1:5, 1:10, 1:20, 1:50, 1:100, 1:200, 1:500, 1:1000
  • Used for large objects like buildings, bridges, machines

C. Enlargement Scale

  • Object is smaller than the drawing
  • Examples: 2:1, 5:1, 10:1, 50:1
  • Used for tiny components like microchips, watch parts, medical devices

Standard Scale Ratios (ISO 5455)

TypeRecommended Scales
Full Scale1:1
Reduction1:2, 1:5, 1:10, 1:20, 1:50, 1:100, 1:200, 1:500, 1:1000
Enlargement2:1, 5:1, 10:1, 20:1, 50:1

Scale Rule

A scale rule (or architect's scale) is a special ruler with multiple scales marked on it, making it easy to measure and draw at different ratios without calculation.

How to Calculate Scale

  • Drawing length = Actual length ÷ Scale denominator
  • Example: Object is 500mm, scale is 1:10
  • Drawing length = 500 ÷ 10 = 50mm

Advantages

  • Allows large or tiny objects to be drawn on standard paper
  • Maintains accurate proportions
  • Makes drawings practical and manageable
  • Universal ratios allow global understanding

Disadvantages

  • Easy to misread scale and make errors
  • Requires careful calculation
  • Changing scale requires redrawing (manual)
  • Small scales can lose fine detail
