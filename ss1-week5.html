📐 SS1 Technical Drawing: Week 5
Drawing Sheet Layout

Learning Objectives: By the end of this lesson, students should be able to set up a drawing sheet with proper margins, border, and title block according to ISO standards.

1. Overview

A drawing sheet layout is the organized arrangement of a technical drawing on paper, following standardized conventions to ensure clarity, professionalism, and easy filing.

Every professional drawing must follow a standard layout so that anyone, anywhere in the world, can read and understand the information presented.

2. Standard Paper Sizes (ISO 216)

Technical drawings use the international "A-series" paper sizes. Each size is half the area of the one above it.

SizeDimensions (mm)Common Use
A0841 × 1189Large maps, posters
A1594 × 841Architectural plans
A2420 × 594Engineering diagrams
A3297 × 420School projects, small plans
A4210 × 297Student assignments, standard printing

Note: SS1 students typically use A4 size for class work.

3. Components of a Drawing Sheet Layout

A. Margin

  • The space between the edge of the paper and the border line
  • Standard margin: 10–20mm on top, bottom, and right sides
  • 20–25mm on the left side (binding side) to allow for filing without damaging the drawing
  • Purpose: Protects the drawing from damage when handling or filing

B. Border

  • A rectangle drawn inside the margin using a thick continuous line (0.7mm – 1.0mm)
  • Defines the actual drawing area where all work is done
  • No drawing should touch or cross the border line
  • Drawn with a 2H or 3H pencil, then inked with 0.7mm pen

C. Title Block

Located at the bottom right corner of the sheet, inside the border. Contains essential information:

  1. Drawing Title – Name of the object/project (e.g., "Hexagonal Bolt")
  2. Drawing Number – Unique reference number (e.g., TD/SS1/001)
  3. Scale – The scale used (e.g., 1:1, 1:2, 2:1)
  4. Date – When the drawing was completed
  5. Drawn By – Name of the student/drafter
  6. Checked By – Name of teacher/supervisor
  7. Sheet Number – e.g., "Sheet 1 of 3" if multiple sheets
  8. School/Company Name and Logo
  9. Projection Symbol – 1st Angle or 3rd Angle projection symbol

Standard Title Block Size for A4: 170mm × 40mm minimum

D. Revision Block

  • Usually placed above the title block or at top right
  • Records all changes made to the drawing after first issue
  • Columns: Revision No., Description of Change, Date, Initials

E. Folding Marks

  • Small triangular marks on the border
  • Guide for folding large sheets (A3, A2, A1) down to A4 size for filing
  • Ensures title block is visible when folded

4. Advantages of Standard Layout

  • Easy identification and filing – Title block gives all info at a glance
  • Professional appearance – Required for WAEC/NECO and industry
  • Legal requirement – Needed for government approvals and contracts
  • Consistency – All drawings from one organization look uniform

5. Disadvantages

  • Takes time to set up manually – Must be drawn before starting the main work
  • Mistakes in title block cause confusion – Wrong date/scale ruins the drawing
  • Different organizations use different formats – Students must adapt
Practical Assignment:
1. On an A4 sheet, draw margins: 20mm left, 10mm top/right/bottom.
2. Draw a border line 0.7mm thick inside the margins.
3. Create a title block 170mm × 40mm at bottom right with all 9 required items.
4. List the 5 standard A-series paper sizes and their dimensions.
