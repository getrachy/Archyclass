📐 SS1 Technical Drawing: Week 10
Introduction to Orthographic Projection

Learning Objectives: By the end of this lesson, students should be able to define orthographic projection, differentiate between 1st and 3rd angle projection, and identify the arrangement of views.

1. Overview

Orthographic projection is a method of representing a 3D object in 2D using multiple views. Each view shows the object from one direction only.

The word "orthographic" means "right-angle drawing" – all projection lines are perpendicular (90°) to the plane of projection.

It is the most important drawing method in engineering and architecture because it shows exact shape and size without distortion.

2. Principal Views

An object can have 6 views, but we normally use 3 principal views:

  1. Front View/Elevation – Shows height and width. Usually the view with most detail.
  2. Plan/Top View – Shows width and depth. Viewed from above.
  3. Side View/End View – Shows height and depth. Can be Left or Right.

3. Types of Orthographic Projection

There are 2 systems used worldwide. Nigeria uses 1st Angle Projection for WAEC/NECO.

A. First Angle Projection (European System)

  • Symbol: Frustum cone with small end facing left
  • Rule: "What you see is drawn on the far side"
  • Object: Placed between observer and plane
  • View Arrangement:
    ViewPosition
    Plan/Top ViewBelow Front View
    Front ViewCentre
    Right Side ViewLeft of Front View
    Left Side ViewRight of Front View
  • Used in: Nigeria, UK, Europe, Asia
1st ANGLE SYMBOL: [◄] (Truncated cone)

B. Third Angle Projection (American System)

  • Symbol: Frustum cone with small end facing right
  • Rule: "What you see is drawn on the near side"
  • Object: Plane is between observer and object
  • View Arrangement:
    ViewPosition
    Plan/Top ViewAbove Front View
    Front ViewCentre
    Right Side ViewRight of Front View
    Left Side ViewLeft of Front View
  • Used in: USA, Canada
3rd ANGLE SYMBOL: [►] (Truncated cone)

4. Key Differences: 1st vs 3rd Angle

Feature1st Angle3rd Angle

Right Side ViewLeft of Front ViewRight of Front View

SymbolSmall end left ◄Small end right ►
UsageNigeria, WAEC/NECOUSA, Canada

5. Rules for Orthographic Drawing

  1. Views must align – Plan directly above/below Front View. Side Views at same height as Front View
  2. Use construction lines – Project from one view to another using 2H pencil
  3. Hidden details – Use dashed lines for edges not visible
  4. Start with Front View – Choose view with most features as Front View
  5. Always include projection symbol – In title block to avoid confusion
  6. Space views evenly – Leave 30-40mm between views on A4

6. Advantages of Orthographic Projection

  • Shows true shape and size – No distortion like perspective
  • Easy to dimension – All features are measurable
  • Universal language – Understood by engineers worldwide
  • Required for manufacturing – Workshops use orthographic drawings

7. Disadvantages

  • Requires imagination – Must mentally assemble 2D views into 3D object
  • Needs multiple views – One view is never enough
  • Takes time to learn – Beginners confuse 1st and 3rd angle
