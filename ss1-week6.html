📐 SS1 Technical Drawing: Week 6
Drawing Lines (Types, Thickness, Uses)

Learning Objectives: By the end of this lesson, students should be able to identify, draw, and state the uses of the 8 standard types of lines used in technical drawing.

1. Overview

Lines are the foundation of technical drawing. Different line types, thicknesses, and styles convey different types of information on a drawing.

Using the wrong line type is like using the wrong word in a sentence – it changes the meaning and causes confusion. All lines must follow ISO standards so they are understood universally.

2. The 8 Standard Types of Lines

3. Detailed Descriptions & Rules

A. Outline/Object Line (Continuous Thick - 0.7mm)

  • Heaviest line on a drawing – Must stand out clearly
  • Defines the visible shape of an object
  • Always drawn first when starting a drawing
  • Must be dense black and uniform thickness

B. Dimension Line (Continuous Thin - 0.35mm)

  • Has arrowheads at both ends touching extension lines
  • Shows the measurement of a feature
  • Dimension figure is placed above the line or in a gap
  • Never crosses other lines if possible

C. Extension/Projection Line (Continuous Thin - 0.35mm)

  • Extends from the object to the dimension line
  • Does not touch the object – leave a small 1mm gap
  • Extends 2-3mm beyond the dimension line

D. Centre Line (Chain Thin - 0.35mm)

  • Alternating long dash and short dash
  • Marks the centre of circles, arcs, and symmetrical features
  • Must extend slightly beyond the feature (2-3mm)
  • Long dashes should intersect at the centre of circles

E. Hidden Detail Line (Dashed - 0.35mm)

  • Short, equally spaced dashes (3mm dash, 2mm gap)
  • Shows edges and outlines not visible from the viewing direction
  • Dashes should start and end touching visible lines

F. Cutting Plane Line (Chain Thick - 0.7mm)

  • Shows where an imaginary cut is made to reveal interior details
  • Has thick arrowheads showing direction of view
  • Lettered A-A, B-B, etc. to match the sectional view

4. Line Thickness Standards

Technical drawing uses only 2 main thicknesses for clarity:

  • Thick Lines: 0.7mm – For outlines and cutting planes only
  • Thin Lines: 0.35mm – For everything else
  • Ratio: Thick lines must be twice as thick as thin lines

5. Advantages of Standardized Lines

  • Universal understanding – Engineers in Nigeria, Japan, and Brazil read the same meaning
  • Reduces confusion – No guessing what a line means
  • Makes drawings cleaner – Hierarchy of line weights improves readability
  • Required for WAEC/NECO – Wrong lines lose marks

6. Disadvantages

  • Requires practice – Mastering 8 line types takes time
  • Poor line quality fails drawings – Wobbly or wrong thickness lines are unprofessional
  • Manual consistency is difficult – Hard to keep all thin lines exactly 0.35mm by hand
Practical Assignment:
1. On A4 paper, draw all 8 types of lines, 100mm long each. Label each type.
2. Use 0.7mm pen for thick lines and 0.35mm pen for thin lines.
3. State the use of: a) Dashed line b) Chain thin line c) Continuous thick line
