Geometrical Constructions – Angles

Learning Objectives: By the end of this lesson, students should be able to construct 30°, 45°, 60°, and 90° angles using compass and ruler without a protractor.

1. Geometrical Constructions – Angles (30°, 45°, 60°, 90°)

Overview

Geometrical construction refers to drawing accurate shapes, lines, and angles using only basic drawing instruments — compass, ruler, and set squares — without relying on a protractor (though protractors can be used for verification).

Why Learn Geometric Construction?

  • Foundation of all technical drawing
  • Develops spatial reasoning
  • Required for drawing complex shapes accurately
  • Used in engineering, architecture, and design

Tools Used

  • Compass
  • Ruler/Straightedge
  • Set squares (30°/60° and 45°)
  • Pencil (H or 2H)

Constructing a 90° Angle

Method 1: Using a Set Square

  • Place the set square on the T-square and draw a vertical line

Method 2: Using a Compass (Perpendicular Bisector)

  1. Draw a baseline AB
  2. Set compass wider than half AB, draw arcs from A and B above and below the line
  3. Connect the intersection points — this gives a 90° angle

Method 3: From a Point on a Line

  1. Mark point P on line AB
  2. Draw arcs on both sides of P
  3. From those arcs, draw two more intersecting arcs above
  4. Connect intersection to P — gives 90°

Constructing a 60° Angle

Method: Using a Compass

  1. Draw a baseline AB
  2. Place compass at A, draw an arc cutting the line at C
  3. With the same radius, place compass at C, draw another arc cutting the first arc at D
  4. Connect A to D — angle DAC = 60°

(This is based on constructing an equilateral triangle — all angles are 60°)

Constructing a 30° Angle

Method: Bisect a 60° Angle

  1. First construct a 60° angle
  2. Bisect it using the angle bisector method:
    • Draw arcs from both arms of the 60° angle
    • Connect the intersection to the vertex
  3. Each half = 30°

Constructing a 45° Angle

Method: Bisect a 90° Angle

  1. First construct a 90° angle
  2. Bisect it:
    • Draw arcs from both arms of the 90° angle
    • Connect intersection to vertex
  3. Each half = 45°

Alternative: Using a 45° Set Square

  • Place the 45° set square on the T-square and draw directly
Assignment:
1. Construct angles of 30°, 45°, 60°, and 90° using compass and ruler only.
2. Verify your constructions using a protractor.
3. Label all construction lines clearly.
