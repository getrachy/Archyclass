📐 SS1 Technical Drawing: Week 9
Introduction to Scales and Scaling

Learning Objectives: By the end of this lesson, students should be able to define scale, state types of scales, and convert actual dimensions to drawing dimensions using a given scale.

1. Overview

A scale is the ratio of the size of the drawing to the size of the actual object.

Most objects are either too large to draw full size (e.g., a house) or too small to see details (e.g., a watch gear). Scaling allows us to represent them proportionally on paper.

Scale = Drawing Size : Actual Size

2. Types of Scales

A. Full Scale (1:1)

  • Drawing size = Actual size
  • Used when object can fit on paper comfortably
  • Example: Small machine parts, bolts, nuts
  • Written as 1:1 or SCALE 1:1

B. Reduction Scale

  • Drawing size < Actual size – Drawing is smaller than real object
  • Used for large objects: buildings, bridges, land
  • Common reduction scales: 1:2, 1:5, 1:10, 1:20, 1:50, 1:100, 1:200, 1:1000
  • Example 1:2 means 1mm on drawing = 2mm on actual object. So 100mm object = 50mm drawing

C. Enlargement Scale

  • Drawing size > Actual size – Drawing is larger than real object
  • Used for tiny objects: watch parts, electronic components, microchips
  • Common enlargement scales: 2:1, 5:1, 10:1, 20:1, 50:1
  • Example 2:1 means 2mm on drawing = 1mm on actual object. So 5mm object = 10mm drawing

3. Standard Recommended Scales (ISO)

CategoryRecommended Scales
Enlargement50:1, 20:1, 10:1, 5:1, 2:1
Full Size1:1
Reduction1:2, 1:5, 1:10, 1:20, 1:50, 1:100, 1:200, 1:500, 1:1000, 1:2000, 1:5000, 1:10000

Rule: Always use standard scales. Avoid 1:3, 1:7, 1:9 because they are hard to calculate.

4. How to Apply Scale

Formula Method:

Drawing Length = Actual Length ÷ Scale Factor

Actual Length = Drawing Length × Scale Factor

Worked Examples:

  1. Example 1: A line is 120mm long. Draw it at scale 1:2
    Drawing Length = 120 ÷ 2 = 60mm
  2. Example 2: On a drawing at scale 1:50, a wall measures 80mm. What is actual length?
    Actual Length = 80 × 50 = 4000mm = 4m
  3. Example 3: A gear tooth is 2mm. Draw at scale 5:1
    Drawing Length = 2 × 5 = 10mm

5. Scale Rules & Scale Drawings

  • Scale Rule: Special ruler with multiple scales marked. Never use it as a normal ruler.
  • Always state the scale in the title block: e.g., "SCALE 1:50"
  • All dimensions on drawing must be actual sizes – Not scaled dimensions. If wall is 4m, write "4000" not "80"
  • Never write "NOT TO SCALE" unless absolutely necessary. It makes drawings useless.

6. Advantages of Using Scales

  • Allows large objects to fit on paper – Can draw a stadium on A4
  • Makes small details visible – Can enlarge tiny watch parts
  • Maintains proportion – All parts reduce/enlarge uniformly
  • Saves paper and storage space

7. Disadvantages

  • Can cause measurement errors – If wrong scale is used
  • Requires calculation – Must convert all dimensions
  • Cannot measure directly from drawing – Must check scale first
Practical Assignment:
1. Define scale and state 3 types of scales with examples.
2. A rectangular field measures 60m × 40m. Draw it at scale 1:500. What are the drawing dimensions?
3. On a 1:20 drawing, a door measures 45mm wide. What is the actual width in metres?
4. Why must you always write dimensions as actual sizes, not scaled sizes?
