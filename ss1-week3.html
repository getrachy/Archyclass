Drawing Sheet Layout & Drawing Lines

Learning Objectives: By the end of this lesson, students should be able to set up a drawing sheet properly and identify/use different types of lines in technical drawing.

1. Drawing Sheet Layout (Borders, Title Block, Margin)

Overview

A drawing sheet layout is the organized arrangement of a technical drawing on paper, following standardized conventions to ensure clarity and professionalism.

Standard Paper Sizes (ISO)

SizeDimensions (mm)
A0841 × 1189
A1594 × 841
A2420 × 594
A3297 × 420
A4210 × 297

Components of a Drawing Sheet Layout

A. Margin

  • The space between the edge of the paper and the border
  • Typical margin: 10–20mm on most sides, 20–25mm on the binding side (left)
  • Protects the drawing from damage at the edges

B. Border

  • A rectangle drawn inside the margin using a thick continuous line
  • Defines the drawing area
  • Standard border line thickness: 0.7mm – 1.0mm

C. Title Block

Located at the bottom right corner of the sheet. Contains:

  • Drawing Title – Name of the object/project
  • Drawing Number – Unique reference number
  • Scale – The scale used (e.g., 1:10)
  • Date – When the drawing was made
  • Drawn By – Name of the drafter
  • Checked By – Name of the reviewer
  • Sheet Number – e.g., Sheet 1 of 3
  • Company/Institution Name and Logo
  • Projection Symbol – 1st or 3rd angle projection

D. Revision Block

  • Records changes made to the drawing
  • Includes revision number, description, date, and initials

E. Folding Marks

  • Small marks to guide folding the sheet to A4 size for filing

Advantages of Standard Layout

  • Easy to identify and file drawings
  • Provides all necessary information at a glance
  • Maintains professionalism
  • Required for official and legal submissions

Disadvantages

  • Takes time to set up manually
  • Mistakes in the title block can cause confusion
  • Not all organizations use the same format

2. Drawing Lines (Types, Thickness, Uses)

Overview

Lines are the foundation of technical drawing. Different line types convey different types of information.

Detailed Descriptions

Outline/Object Line (Thick Continuous)

  • Heaviest line on a drawing
  • Defines the visible shape of an object
  • Always drawn first

Dimension Line (Thin Continuous)

  • Has arrowheads at both ends
  • Shows the measurement of a feature
  • Never breaks for a dimension figure in modern standards

Extension/Projection Line (Thin Continuous)

  • Extends from the object to the dimension line
  • Does not touch the object directly (small gap left)

Centre Line (Chain Thin)

  • Alternating long and short dashes
  • Marks the centre of circles, arcs, and symmetrical features
  • Extends slightly beyond the feature

Hidden Detail Line (Dashed)

  • Short, equally spaced dashes
  • Shows edges and outlines that are not visible from the viewing direction

Section/Cutting Plane Line

  • Shows where an imaginary cut is made to reveal interior details
  • Has arrows showing the direction of view

Advantages of Standardized Lines

  • Universal understanding across different countries/firms
  • Reduces confusion and misinterpretation
  • Makes drawings cleaner and more readable

Disadvantages

  • Requires practice to master all line types
  • Poor line quality makes drawings unreadable
  • Manual consistency is difficult to achieve
Assignment: Draw an A4 sheet layout with proper margins, border, and title block. Then practice drawing all 8 types of lines with correct thickness.
