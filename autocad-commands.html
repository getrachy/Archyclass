1. Draw Commands

Foundation for SS1. Create basic geometry. Master these first.

LINE L
+
Creates straight line segments. Most used command in TD.
Command: LINE or L
Specify first point: Click or 0,0
Specify next point: Click or @100,0
How to use for WAEC:
  1. Type L → Enter
  2. Click start point or type coordinates like 50,50
  3. For horizontal: Type @100,0 → Enter. For vertical: @0,100
  4. Press Enter to end, or C to close shape
Pro Tip: Use ORTHO F8 to force 90° lines. Critical for orthographic projections in SS3.

OFFSET O
+
Creates parallel lines/curves at specified distance. Essential for wall thickness.
Command: OFFSET or O
Specify offset distance: 5
Select object: Click line
Specify side: Click
SS2 Application:
  1. Draw outer wall line first
  2. Type O → Enter → Type 9 → Enter for 225mm wall
  3. Click wall line → Click inside for inner wall
Pro Tip: Use for section hatching boundaries. Offset 3mm for concrete slab thickness.
WAEC SS2 Week 6
ARRAY AR
+
Creates multiple copies in rectangular or polar pattern. SS3 time saver.
Command: ARRAYRECT or AR
Select objects → Enter
Columns: 5, Rows: 1, Spacing: 3000
WAEC SS3 Use Case:
  1. Draw one window
  2. AR → Select window → Enter
  3. Set Columns: 4, Between: 1200 for 4 windows at 1.2m c/c
Pro Tip: ARRAYCLASSIC for old dialog. Use ARRAYPOLAR for circular stairs.

4. Layers + Properties

SS3 Professional Practice. WAEC marks you down if you don't use layers.

LAYER LA
+
Manages layers. Think of layers as transparent sheets. Walls on one, dimensions on another.
Command: LAYER or LA
New Layer: WALL, TEXT, DIM, HATCH
WAEC Layer Standard:
  1. WALL - Continuous, 0.5mm, White
  2. DIM - Continuous, 0.25mm, Red
  3. HATCH - Continuous, 0.18mm, Yellow
  4. TEXT - Continuous, 0.35mm, Cyan
Pro Tip: Freeze DIM layer before plotting if teacher wants no dimensions. Turn off HATCH to see under it.
